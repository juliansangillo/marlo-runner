A delegate function that will fire the StateChangeSignal when a callback is recieved from the IInfo. This will happen when the value of one of its keys was modified.

Parameters
idThe unique identifier (the gameObject tag) that fired the signal so the subscribers to the signal know which info object had their state changed
kThe key that uniquely identifies the value that changed
vThe new value of the key. May be of any type
