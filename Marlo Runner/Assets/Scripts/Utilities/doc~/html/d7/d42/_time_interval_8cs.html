enum TimeInterval
strong

An enumeration of seconds, minutes, and hours. Since enumerations are fundamentally integers, you can multiply some time already in seconds by one of these to convert it to that interval.

Seconds - Default interval
Minutes - Interval representing 60 seconds
Hours - Interval representing 3600 seconds

Example:

float timer = 5 * TimeInterval.Minutes;
timer -= Time.deltaTime;
if(timer <= 0f)
Debug.Log("This ran for 5 minutes, not 5 seconds.");
